## Install
```
npm i anygate -g
```

## Usage
```bash
# use current folder as the root directory of the static resource server
anygate

# set the port number as 1234
anygate -p 1234

# set the host as 'localhost'
anygate -h localhost

# set the root directory as '/usr'
anygate -d /usr
```
